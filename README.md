# 📊 E-commerce Sales Analysis

This project analyzes an e-commerce dataset to extract meaningful insights about sales, profit, and customer behavior using Python.

## 🚀 Project Overview

The goal of this project is to perform data analysis and visualization to understand:

- Monthly sales trends
- Highest and lowest performing months
- Sales distribution across categories and sub-categories
- Profit analysis by month, category, and segment
- Sales-to-profit ratio

## 🛠️ Tools & Technologies

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📈 Key Analysis Performed

### 1. Monthly Sales Analysis
- Calculated total sales per month
- Identified highest and lowest sales months

### 2. Category Analysis
- Compared sales across different product categories
- Found highest and lowest performing categories

### 3. Sub-Category Analysis
- Deep dive into sub-category performance

### 4. Profit Analysis
- Monthly profit trends
- Highest profit month identification

### 5. Customer Segment Analysis
- Sales and profit comparison across segments

### 6. Sales-to-Profit Ratio
- Evaluated business efficiency

## 📊 Visualizations

- Bar charts for monthly sales
- Category-wise sales comparison
- Profit analysis graphs
- - Pie Chart for Category Analysis

## 📌 Key Insights

- Identified peak sales month
- Found most profitable category
- Discovered underperforming segment
